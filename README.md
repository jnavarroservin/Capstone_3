# Capstone 3: Power BI Sales Analysis – West Region

## Project Overview

This capstone project was completed as part of the Year Up United Data Analytics Training Academy Week 12 Power BI capstone.

The goal of this project was to clean, model, analyze, and visualize sales data for EmporiUm, a virtual student bookstore that sells books, technology, and student-related products through both in-store and online sales channels.

For this analysis, I focused on the **West Region**, managed by **Jim Heck**, to identify sales trends, category performance, geographic performance, and top-selling books.

---

## Business Objective

The purpose of this project was to provide actionable business insights to support better decision-making for the regional sales team.

Key questions explored:

* How has sales performance changed over time?
* Which product categories generate the most revenue?
* Which states contribute the most sales in the West region?
* Which books are the top-selling titles in the region?

## Data Preparation

The project began with cleaning and transforming multiple datasets in Power Query, including:

* Store Sales
* Online Sales
* Products
* Store Locations
* Shipping Data

Key preparation steps included:

* Correcting data types
* Removing unnecessary columns
* Handling null values and errors
* Building table relationships
* Creating a Date table for time-based analysis
* Creating DAX measures for key sales metrics

## Key Insights

This analysis highlights important sales trends and opportunities within the West region.

Insights include:

* Sales performance trends over the four-year period
* Highest-performing product categories
* Strongest-performing states in the West region
* Best-selling books and customer purchasing patterns

These insights can help support better decisions around inventory planning, marketing strategy, and regional sales growth.
